Sa-BOT-age
Tools Used: Python, Solidworks, 3D-Printing
Objective: Design, assemble, and program a robot that is able to walk.

Method: I virtually prototyped my design of a walking robot, which is inspired by how crabs walk. I then 3D-printed and assembled the components. I also attached/wired four LewanSoul servo motors and a camera, with which I interface via a Python program running on a Raspberry Pi. It has three static legs that give it a stable resting position, and two dynamic arms to pull it forward.

Additional Info: I worked on this robot as part of my Robotics Studio Course.
